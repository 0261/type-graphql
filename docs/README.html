README

Documentation

This folder consist of series of docs showing how to use TypeGraphQL to create GraphQL's schema. You can learn both the basics as well as advanced techniques and additional features.

FAQ - Frequently Asked Questions

First steps

  • introduction (what is TypeGraphQL and why it was made)
  • getting started (creating simple example app with TypeGraphQL)

Basics

  • types and fields (how to create object type, define its fields and types)
  • queries, mutations and field resolvers
  • app bootstrapping (how to build the TypeGraphQL's schema and run express app)

Advanced

  • scalars (built-in and custom scalars)
  • enums (basic enum usage)
  • unions (defining type-safe union types)
  • interfaces and inheritance (creating GraphQL's interfaces and reuse input/args definitions)
  • subscriptions (creating GraphQL subscriptions with pubsub and filters)

Features

  • dependency injection (usage of IoC containers)
  • authorization (restricting access to fields or queries/mutation, based on roles)
  • validation (nice integration with class-validator)
